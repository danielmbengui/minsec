# MINSEC Landing Page — Next.js App Router

Projet Next.js en JavaScript, bilingue FR/EN, avec animations CSS et identité blockchain/sécurité/minage.

## Installation

```bash
npm install
npm run dev
```

Puis ouvrir `http://localhost:3000`.

## Structure

- `app/page.js` : landing page principale bilingue
- `app/globals.css` : design, responsive, animations
- `app/layout.js` : metadata
- `public/minsec-logo.jpeg` : logo fourni

## Déploiement

Compatible Vercel, serveur Node, ou export statique si besoin.
